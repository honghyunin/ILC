# Back-End 백엔드 공부

## 백엔드의 정의

- 백엔드는 소프트웨어 개발 프로세스에서 서버 관련을 개발하는 분야이다.데이터를 다루며 클라이언트에서 원활하게 작동을 할 수 있게 합니다.

- 백엔드는 프론트엔드에 있는 유저들이 취하는 행동을 합니다.
