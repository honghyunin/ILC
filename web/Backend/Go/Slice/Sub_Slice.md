# 부분 슬라이스

- 슬라이스의 일부를 빼내어 부분 슬라이스를 만들 수 있습니다.
- 부분 슬라이스는 ``Slice[First index, Last Index]`` 형식으로 만듭니다.
```go
    s := []int(0,1,2,3,4,5)
    s:= s[2:5]
    fmt.Println(s)
```

- 슬라이스는 인덱스의 처음과 마지막 둘 중 하나 혹은 둘 다를 생략할 수도 있습니다. 처음 인덱스가 생략되면 0 이, 마지막이 생략되면 마지막 인덱스가 자동으로 대입됩니다.
``` go
s := []int(0,1,2,3,4,5)
s = s[2:5] // 2, 3, 4
s = s[1:] // 3, 4
fmt.Println(s) // 3, 4 출력
```
